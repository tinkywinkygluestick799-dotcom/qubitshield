# QubitShield

Shor's 9-qubit vs Steane's 7-qubit quantum error correction codes.

Result: Steane's code outperforms Shor's by 2x.

Run:
git clone https://github.com/tinkywinkygluestick799-dotcom/qubitshield
cd qubitshield
pip install -r requirements.txt
python compare_codes.py

Dependencies: Python 3.8+, Qiskit, NumPy, Matplotlib
